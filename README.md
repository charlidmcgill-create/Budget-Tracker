# Budget-Tracker
Full-stack budget tracker with REST API and CSV imports

## Features
- user authentication
- import transactions from CSV bank statements
- automatic transaction categorization
- monthly spending summaries
- budget limits by category

## Architecture
- frontend communicates with backend via REST API
- backend handles authentication, data validation
- PostgreSQL database stores users, transactions, and categories
